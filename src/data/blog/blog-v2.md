心血来潮，迁移到了新的博客模版。

最早应该是高中时开始写博客，现在想来已经有十年已久。虽然时间跨度大，但其实中间没咋折腾过，可能也是因为我懒。一开始用了 Leanote 的订阅制的发布服务（刚才搜了下，没想到还活着），之后转到了 GitHub Pages，用的 [kitian616/jekyll-TeXt-theme](https://github.com/kitian616/jekyll-TeXt-theme)，现在换到了 [satnaing/astro-paper](https://github.com/satnaing/astro-paper)。

十年很快，世界变了太多。我现在的生活，十年前，哦不，五年前，哦不，两年前，一年前的自己能想象的到吗？

说来惭愧，我虽然曾经是个程序员，但真是完全错过了最近一年 AI 编程的风潮。什么 Cursor、Claude Code 云云，统统没用过。这次用 Gemini CLI 迁移博客，还是第一次尝试 Agent 编程。真是又惊又喜，感叹它能完美完成我给它的任务，更感叹有的时候它还能额外完成我没想到的事。原本以为需要一个周末的迁移，三四个小时就搞定了。也感受到了一种久违的想要造东西的欲望。

这次换模版是因为上个模版实在是蹩脚：一是自己从来没用过 Ruby，懒得搭本地环境；二是自己也不懂 Ruby，想搞点自定义要学很多东西。一想到这些事情就头疼，想必也是自己变得懒得写博客了的原因之一。

上次岛波海道之行回来，一直想模仿一下 BYVoid，写个游记之类的东西，就是因为看不惯旧的博客模版，迟迟没有开写
！希望自己从现在开始能多留下点文字呀！

最后，我还特意部署了一份上个模版的博客，供自己和感兴趣的人来怀旧一下：https://okamiwong.github.io/blog-v1-public
